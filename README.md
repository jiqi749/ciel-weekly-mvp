# Ciel · 一周怎么过

一个纯静态、local-first 的个人时间与注意力管理 MVP。

它不要求记录全天流水，核心是：

- 每周先保护 3 段作品集产出时间；
- 提前安排并确认认真陪伴和攀岩时间；
- 临时 Todo 先进入收件箱，不自动挤进今天；
- 作品集完成时留下代码、文档或问题解决证据；
- 加班等意外发生后重新安排，不用睡眠填补冲突；
- 出差、高压或恢复期可以切换为特殊周。

## 数据

数据保存在当前浏览器的 `localStorage`，无需账号和后端。右上角“备份”可以导出 JSON 文件。

目前不支持跨设备自动同步，换设备或清理浏览器数据前需要先备份。

## 运行

这是一个无需构建的静态网站，入口文件为 `index.html`，可部署到任意静态托管服务。
